# DEXABID

> Deploy Prediction Trading Agents in Minutes.

Dexabid is an AI-powered automation platform for prediction markets on Solana. Configure your trading strategy, define risk parameters, deploy autonomous agents, and let them learn from every prediction round.

Whether you're an experienced trader or just getting started, Dexabid gives you intelligent automation without sacrificing manual control.

# 🚀 What is Dexabid?

Prediction markets move fast.

Monitoring every event, adjusting positions, and managing risk manually can quickly become overwhelming.

Dexabid introduces autonomous trading agents that continuously monitor prediction rounds, execute predefined strategies, and improve over time using historical performance and configurable trading rules.
